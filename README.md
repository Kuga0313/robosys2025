## robosys2025

# plusコマンド
![test](https://github.com/Kuga0313/robosys2025/actions/workflows/test.yml/badge.svg)

標準入力から読みんだ数字を足していく。  
（例）入力した値　：５  
　　　１＋２＋３＋４＋５より  
　　　出力される値：１５  

## 使い方

$ cd robosys2025  
$ seq 5 | ./plus

## 必要なソフトウェア
- Python  
  - テスト済みバージョン: 3.7〜3.10

## テスト環境
- Ubuntu 24.04 LTS
